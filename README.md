QuickPic
========

Simple cross-platform picture viewer; comparable to Windows' default viewer.

* Rotate (auto-saved)
* Zoom
* Prev/Next (siblings)
* Delete
* Group on Desktop (easy uploading for the mrs.)

Requires
--------
* Qt 5
